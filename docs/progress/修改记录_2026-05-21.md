## 23:30 UTC · Tick 3 (执行组) — Textual TUI + 6 导出器插件 + 配置中心

**改了什么**
- 配置中心 (`backend/app/config/` 包重构 - 替换原 config.py 单文件):
  - `settings.py`: pydantic-settings, 新增 11 个配置字段 (export/MySQL/MongoDB/Webhook), 总计 27 项
  - `loader.py`: YAML load/save + `merge_into_env` 把 dict 注入 os.environ 供 pydantic-settings 读
  - `schema.py`: FIELD_GROUPS 9 组分组 + SECRET_FIELDS 集合 (3 项敏感) + JSON Schema 导出供 UI 渲染

  - `registry.py`: 装饰器风格自注册 (`@register_exporter("xxx")`)
  - 6 内置实现: csv (utf-8-sig BOM + 自定义分隔符), json (嵌套含 meta), sqlite (独立 db 文件 + REPLACE INTO), mysql (aiomysql + REPLACE INTO), mongodb (motor + replace_one upsert), webhook (httpx POST + Bearer token)
- Textual TUI (`cli/tui/`):

  - `screens/`: main (sidebar 6 项菜单) + tasks (CrawlTask 列表 + r 刷新) + config (按 group 渲染只读视图 + 敏感字段脱敏) + logs (RichLog 静态读取) + export (Select 6 格式 + Input UID/limit + 一键导出) + help (cheatsheet)
- CLI 扩展 (`cli/commands.py`):
  - `wcn tui` 实接 WCNApp().run()
